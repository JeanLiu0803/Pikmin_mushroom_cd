# Pikmin 蘑菇倒數計時器

一個可愛、乾淨、行動裝置友善的 Pikmin 風格蘑菇倒數計時器。使用者可以為多個蘑菇戰設定剩餘分鐘，系統會自動加上可調整的緩衝時間，方便預留回到遊戲查看結果的時間。
網頁位置：https://jeanliu0803.github.io/Pikmin_mushroom_cd/

## 功能

- 新增蘑菇位置或名稱，未輸入時預設為「蘑菇」
- 輸入剩餘時間：分鐘
- 開始倒數時自動加上緩衝時間
- 預設緩衝時間為 4 分鐘
- 可在頁面底部設定緩衝時間
- 可設定完成計時後卡片消失的時間，預設為 15 秒
- 使用 `localStorage` 記住緩衝時間與完成後消失時間設定
- 支援多個倒數計時器同時運行
- 目前進行中的倒數會顯示在新增計時表單上方
- 目前進行中的倒數會以固定大小圖卡形式顯示
- 圖卡背景會從 `images/card/` 隨機挑選
- 圖卡會依照完成時間排序，越快結束越前面，已完成的卡片會排到最後
- 每張圖卡顯示蘑菇名稱與目前倒數時間
- 倒數時間顯示為「幾分幾秒」
- 倒數結束後，卡片會依設定保留一段時間再自動移除
- 可手動刪除單一計時器
- 首頁右側圖片使用 `images/Pikmin_cover.png`
- RWD 響應式版面，適合桌機與手機瀏覽器

## 如何使用

直接用瀏覽器打開 `index.html` 即可使用。

也可以在專案資料夾啟動簡單的本機伺服器：

```bash
python3 -m http.server 8000
```

接著打開：

```text
http://localhost:8000
```

## 操作方式

1. 在「蘑菇位置 / 名稱」輸入蘑菇名稱或地點。
2. 輸入遊戲中顯示的剩餘分鐘。
3. 按下「開始倒數」。
4. 計時器會以「剩餘分鐘 + 緩衝時間」開始倒數。
5. 若想調整緩衝時間或完成後卡片停留秒數，可到頁面底部的「設定」修改。

## 緩衝時間

緩衝時間預設為 4 分鐘。每次新增計時器時，會使用當下設定的緩衝時間。

例如：

```text
輸入剩餘分鐘：10 分
緩衝時間：4 分
實際倒數：14分 00秒
```

緩衝時間設定會保存在瀏覽器的 `localStorage`，下次開啟同一個瀏覽器時會自動套用。

## 完成後卡片消失時間

完成後卡片消失時間預設為 15 秒。倒數完成時，該圖卡會先標示為 `finished`，並自動排序到列表最後，再依設定時間自動移除。

這個設定同樣會保存在瀏覽器的 `localStorage`。

## 目前限制

- 計時器本身不會保存到 `localStorage`，重新整理頁面後會清空目前倒數。
- 沒有通知音效或系統通知。
- 圖卡背景圖片清單目前寫在 `script.js` 的 `CARD_BACKGROUNDS`。

## 開發筆記

- `index.html`：頁面結構、表單、計時器圖卡模板與設定區。
- `style.css`：Pikmin inspired 視覺風格、固定大小圖卡、首頁圖片與響應式排版。
- `script.js`：表單處理、倒數邏輯、多計時器排序、隨機圖卡背景、完成後自動移除與 `localStorage` 設定保存。
